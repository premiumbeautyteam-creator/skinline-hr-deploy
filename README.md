# SkinLine HR Deploy

Bootstrap-репозиторий для разворачивания HR CRM на VPS Timeweb.

## Использование
На свежем сервере под root:
```
curl -fsSL https://raw.githubusercontent.com/premiumbeautyteam-creator/skinline-hr-deploy/main/bootstrap.sh | bash
```

Это приватный репозиторий, используется только для одноразовой передачи кода на VPS.
